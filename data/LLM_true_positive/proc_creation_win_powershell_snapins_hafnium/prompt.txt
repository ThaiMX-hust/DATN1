Windows process creation logs, and command-line parsing.

Task:
Generate realistic, benign command-line examples that would produce true-positive
Windows process creation events for the Sigma rule below. The commands are for
authorized lab validation and regression testing only.

Requirements:
- Output command lines only, one per line. Do not include explanations, numbering,
  bullets, code fences, or any extra text.
- Each command must meaningfully match at least one detection path in the Sigma rule.
- Prefer safe placeholders and harmless actions, such as TEST-HOST, example.com,

  persistence, evade detection, exfiltrate data, or cause damage.
- Include variations only when they exercise materially different rule conditions,

- If a binary is not available by default on Windows, assume it is installed at the
